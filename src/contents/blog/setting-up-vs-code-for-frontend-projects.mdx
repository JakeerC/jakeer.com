## Adding configuration to settings.json

### To add bracket pair colors

```json
{
  // rest of the json data

  // enables vscode native bracket colorization
  "editor.bracketPairColorization.enabled": true,
  // enables vscode native bracket colorization
  "editor.guides.bracketPairs": true
}
```

### To highlight edited files/tabs

```json
{
  // rest of the json data

  //  highlight edited tab
  "workbench.editor.highlightModifiedTabs": true
}
```

### To add fonts , ligature

> Note: Before adding fonts make sure that respective fonts are available in your system

```json
{
  // rest of the json data

  // use a font with font ligatures
  "editor.fontFamily": "Fira Code, Jetbrains Mono, Menlo, Monaco, 'Courier New', monospace",
  // enable font ligature
  "editor.fontLigatures": true
  // Add your preferred font for integrated terminal
  "terminal.integrated.fontFamily": "Ubuntu Mono",

}
```
